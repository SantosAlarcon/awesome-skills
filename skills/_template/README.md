# Skill Template

Use this template when creating a new skill.

## Structure

```
category/skill-name/
├── SKILL.md          # Required
├── examples/         # Optional
├── prompts/          # Optional
└── resources/        # Optional
```

## Checklist

- [ ] SKILL.md with all required sections
- [ ] Clear, descriptive title
- [ ] At least one practical example
- [ ] Checked for spelling and grammar
- [ ] Follows contribution guidelines
